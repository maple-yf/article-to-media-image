# Article to Media Image

> 文章转分享卡片 — 将文章/笔记转化为精美 HTML 卡片图片，支持多种视觉风格。可用于小红书、微信公众号贴图、公众号文章插图等社交媒体场景。

## 项目简介

这是一个 AI Agent 技能（Skill），用于将文章内容自动转化为适合社交媒体分享的精美卡片图片。可以用于主流Agent，包括OpenClaw、OpenCode、Claude Code等。AI 会分析文章内容，提取核心信息，并根据内容调性选择最匹配的视觉风格，最终生成 HTML 卡片并转换为图片。

## 核心特点

### 🎨 内容决定形式
AI 不是模板填充机，而是设计师。根据文章内容自由设计视觉风格：
- **严肃分析** → 深色/衬线风格（商务深蓝风、杂志编辑风）
- **轻松科普** → 手账/波普风格（孟菲斯波普风、北欧极简风）
- **技术教程** → 终端/蓝图风格（暗色科技风、信息图蓝图风）

### 📱 手机优先
图片专为手机分享优化，缩放 3 倍后依然清晰可读：
- 固定宽度 1080px
- 最小字号：正文 30px，标题 72px
- 最多 2 栏布局，确保移动端体验

## 工作流程

```
用户给文章
    ↓
① 内容分析 — 提取核心论点、关键数据、对比维度
    ↓
② 自由设计 — 根据内容调性，自主决定视觉风格、配色、布局
    ↓
③ 生成 HTML — 遵守技术底线规范直接生成
    ↓
④ 后处理 — 运行 scripts/post-process.sh（可选）
    ↓
⑤ 截图交付 — 在浏览器中打开 HTML 并截图
```

## Agent 使用方式

### 作为 Claude Skill 使用

1. **安装 Skill**
   - 将项目克隆到本地
   - 在 Claude Code 中配置 Skill 路径

2. **触发场景**
   以下任一场景会自动触发本技能：
   - "帮我生成文章分享卡片"
   - "把这篇笔记做成信息图"
   - "文章转图"
   - "生成分享图"
   - "内容卡片化"
   - "文章可视化"

3. **输入格式**
   ```
   [文章内容或链接]
   可选：指定风格（如"用孟菲斯风格"）
   ```

4. **输出**
   - HTML 文件（可在浏览器中打开）
   - PNG 图片（通过浏览器截图）

### 手动使用流程

```bash
# 1. 生成 HTML（由 AI Agent 完成或手动编写）
# 2. 后处理（可选，修复常见问题）
./scripts/post-process.sh output.html

# 3. 在浏览器中打开
open output.html

# 4. 截图
# - macOS: Cmd+Shift+4
# - Windows: Win+Shift+S
```

## 技术规范

生成 HTML 时**必须遵守**的硬性约束：

| 约束项 | 要求 | 原因 |
|--------|------|------|
| 宽度 | 固定 1080px | 手机截图适配 |
| 容器高度 | min-height，不用固定 height | 防止内容被截断 |
| 正文字号 | 最小 30px | 手机缩放 3x 后可读 |
| 标题字号 | 最小 72px | 视觉层级清晰 |
| 布局栏数 | 最多 2 栏 | 移动端体验 |
| 模块数量 | 4-5 个，不超过 6 个 | 信息密度合理 |

详细规范请参阅 [rules/01-技术底线.md](rules/01-技术底线.md)

## 设计理念

**你是设计师，不是模板填充机。**

- **信息精炼** — 一张图讲清一个核心论点 + 3-4 个支撑模块
- **每次不同** — 配色、字体、布局每次都应该有变化
- **视觉平衡** — 避免千篇一律，根据内容自由设计
