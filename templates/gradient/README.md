# Gradient 模板

## 设计特征

Gradient模板采用温暖的渐变背景与现代玻璃拟态（Glassmorphism）设计相结合，创造出柔和、优雅的视觉体验。

## 适用场景

- 温暖情感类文章分享
- 灵感语录和名言展示
- 个人博客和心情记录
- 创意写作和散文分享
- 品牌故事和案例展示

## 关键设计点

### 配色方案

渐变背景采用4种温暖的渐变色调，自动轮换：

- **Orange（橙色）**: `#FFEDD5 → #FED7AA → #FDBA74`
- **Purple（紫色）**: `#FAE8FF → #F0ABFC → #E879F9`
- **Green（绿色）**: `#ECFDF5 → #A7F3D0 → #6EE7B7`
- **Blue（蓝色）**: `#E0F2FE → #7DD3FC → #38BDF8`

文本颜色层级：
- 主文本: `#1E293B`
- 次要文本: `#475569`
- 辅助文本: `#64748B`

### 字体

- 系统字体栈：`-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Noto Sans SC', sans-serif`
- 代码字体：`'Courier New', monospace`

### 布局

- **卡片比例**: 3:4竖版设计
- **卡片尺寸**: 最大宽度480px
- **内边距**: 48px
- **圆角**: 24px
- **居中对齐**: 完美居中的视觉体验

## 技术要点

### 玻璃拟态效果

- 半透明白色背景: `rgba(255, 255, 255, 0.85)`
- 背景模糊: `backdrop-filter: blur(12px)`
- 柔和边框: `1px solid rgba(255, 255, 255, 0.5)`
- 层次阴影: 双层阴影营造深度感

### 内容类型支持

1. **Title（标题）**: 42px大标题 + 渐变分隔线
2. **Content（内容）**: 18px正文，1.8行高
3. **Quote（引用）**: 大号引号标记 + 20px斜体
4. **Highlight（重点）**: 24px加粗强调
5. **Code（代码）**: 白色卡片包裹，等宽字体

### 视觉元素

- **序号徽章**: 48px圆形徽章显示段落序号
- **底部圆点**: 三个装饰性圆点，增加设计感
- **渐变方向**: 135度对角渐变，营造动感
