理解两个物理约束

Agent有两个像光速一样不能许愿消失的物理约束：上下文容量和注意力带宽。

// 物理约束不是bug
const constraints = {
  capacity: "上下文有限，塞得越多性能越差",
  bandwidth: "注意力零和，多任务互相干扰"
};
2

识别战场：OS vs Application

OS的命题是WHAT（做什么），Application的命题是HOW（怎么做）。垂类不该做OS，该做Application。
