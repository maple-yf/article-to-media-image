E-08

三层光谱对比

形态 执行方式 壁垒
Prompt 指令 Agent自己推理 壁垒为零
Script 工具 外部代劳 壁垒低
Application 代劳+状态+设施 壁垒高
04
F-15

核心洞察

OS的命题是WHAT（做什么）——在有限容量和带宽下帮用户完成尽可能多的事。

Application的命题是HOW（怎么做）——每次被调用时提供最大领域价值。

速查

  • 上下文容量
  • 注意力带宽
  • 领域状态 ≠ Memory
  • 越用越厚
