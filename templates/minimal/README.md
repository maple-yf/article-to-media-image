# Minimal（北欧极简）模板

## 设计特征
- 大量留白，呼吸感强
- 衬线字体 (Playfair Display + Lora)
- 灰度配色，偶尔单色点缀
- 无装饰元素，纯粹排版

## 适用场景
- 技术文档
- 严肃分析
- 学术内容

## 关键设计点
- **配色**:
  - 背景: #FAFAFA 或 #FFFFFF
  - 文字: #2C2C2C
  - 强调: #6B7280
- **字体**:
  - 标题: Playfair Display
  - 正文: Lora
  - 衬线字体增强专业感
- **布局**:
  - 单栏布局
  - 模块间距 32px
  - 内边距 48px

## 技术要点
- 固定宽度 1080px
- 使用 min-height 而非固定 height
- 字号：标题 72px，正文 32px
