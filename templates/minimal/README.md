# Minimal（北欧极简）模板

## 设计特征
- 大量留白，呼吸感强
- 系统无衬线字体 (-apple-system + 'Segoe UI' + 'Noto Sans SC')
- 蓝色系配色，清新现代
- 无装饰元素，纯粹排版

## 适用场景
- 技术文档
- 严肃分析
- 学术内容

## 关键设计点
- **配色**:
  - 背景: #FFFFFF (主) / #F8FAFC (次)
  - 文字: #0F172A (主) / #475569 (次)
  - 强调: #3B82F6 (蓝色)
  - 边框: #E2E8F0
- **字体**:
  - 系统无衬线字体栈
  - 跨平台原生体验
  - 中文支持 ('Noto Sans SC')
- **布局**:
  - 单栏布局
  - 容器最大宽度 420px
  - 卡片内边距 60px/40px
  - 卡片间距 24px

## 技术要点
- 最大宽度 420px
- 固定宽高比 3:4 (aspect-ratio)
- 字号：标题 42px，正文 18px
- CSS变量管理配色系统
