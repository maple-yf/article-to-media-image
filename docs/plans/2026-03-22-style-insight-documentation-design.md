# 03-风格灵感.md 文档重写设计

## 设计概述

本文档记录了对 `rules/03-风格灵感.md` 的重新设计和内容填充工作，旨在为AI提供完整的14种模板风格描述，以便在生成文章配图时自动选择最合适的风格。

## 背景

原有的 `03-风格灵感.md` 文档仅描述了5种风格（minimal、gradient、card、dark、tech_modern），但这些风格对应的模板文件在 `templates/` 目录中并不存在。实际目录中有14个HTML模板文件，需要逐一分析并描述其风格特征。

## 设计目标

1. **完整性**：覆盖全部14个模板文件的风格描述
2. **结构化**：每个风格包含统一的描述维度，便于AI解析
3. **可操作性**：提供AI匹配关键词和选择建议
4. **详细性**：包含视觉特征、配色、字体、布局等完整信息

## 文档结构设计

### 1. 头部元数据

```markdown
<!--
input: 文章内容
output: 14种模板的风格说明，供AI选择时参考
pos: 风格选择参考
-->
```

### 2. 每个风格的描述维度

每个模板风格包含以下9个维度：

| 维度 | 说明 | 示例 |
|------|------|------|
| 风格名称 | 中文风格名称 | 信息图蓝图风 |
| 参考文件 | 对应的HTML文件 | templates/infographic.html |
| 视觉特征 | 核心视觉元素和布局特点 | 蓝图网格背景、模块卡片 |
| 配色方案 | 主色调、辅助色、背景色 | 背景: #F2F2F2 |
| 字体 | 使用的中英文字体 | 英文: SF Mono |
| 布局特点 | 模块组织方式 | 3列网格、双栏布局 |
| 适用场景 | 适合的文章类型 | 技术对比、产品评测 |
| AI匹配关键词 | 便于AI识别的关键词 | `对比` `评测` |

## 设计决策

### 1. 详细程度选择

用户确认选择**详细描述**，包含：
- 完整的视觉特征描述
- 具体的配色方案（含色值）
- 字体信息（英文字体名称）
- 布局特点说明
- 适用场景描述
- AI匹配关键词列表

### 2. 用途定位

文档主要用途是**供AI选择模板时参考**，因此：
- 强调结构化描述，便于机器解析
- 提供关键词标签，支持自动匹配
- 包含选择建议表，简化决策逻辑

### 3. 命名规范

- 风格名称采用中文描述性命名
- 文件名保持与templates目录一致
- 色值使用HEX格式（#RRGGBB）
- 字体名称保持原始格式
